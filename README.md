# 🌌 Vibe Vault

**Made a game with Claude? It lives here.**

Vibe Vault is a community home for vibe-coded games — games people built by describing what they wanted and shaping it with an AI, prompt by prompt. Right now those games get posted once and vanish into a feed. Here they stay: playable in your browser, one click, no installs.

🎮 **Play:** every game in the vault runs right in your browser.
📤 **Share:** built something with Claude (or another model)? Submit it — from a one-prompt jam toy to a game you've iterated on for months. Both belong here.
💬 **Rate & discuss:** each game has its own discussion thread for reactions, ratings, and feedback to the creator.

---

## 🕹️ Play the games

Visit the vault: **the catalogue page lists every game** — click a card to play. Games live in this repo under [`/games/`](games/), one folder each.

---

## 📤 Submit your game

Two ways to be in the vault:

### Open games (the default)
Your game's source lives in this repo, free for anyone to play and learn from.

1. Your game should be a **self-contained browser game** — ideally a single `index.html` (inline JS/CSS), or a small folder with no build step and no external network calls.
2. **Open a [game submission issue](../../issues/new/choose)** with your game attached or linked — or send a pull request adding `games/<your-game-name>/index.html`.
3. Tell us: the game's name, a one-line description of its vibe, and which AI helped you build it.
4. The curator plays everything before it's merged. If it runs and it's yours, it's in.

### Creator listings (paid games)
Built something bigger you want to charge for? Submit a **playable demo** for the vault plus a link to the full version on [itch.io](https://itch.io) (or your own page). The vault is your discovery layer; payment happens entirely on your side — we never touch the money.

**Submission rules (both tiers):**
- ✅ All assets (art, music, code) are yours, AI-generated by you, or properly licensed — you confirm this on submission.
- ✅ Browser-playable, no installs, no accounts, no tracking.
- ❌ No malware, no crypto-miners, no data collection. Submissions are reviewed before merge.

---

## 🧰 Run locally

```bash
git clone https://github.com/Vibe-Vault/Vibe-Vault.git
cd Vibe-Vault
```
Open `index.html` in any modern browser. No build step.
